# The server
> A boilerplate for web server, with modular architecture over microservices approach.

## How to start
```
$ npm install
```

## How to start
```
$ npm start
```

## How to test
```
$ npm test
```

## How to test, finding coverage, build, running lint, docs everything
```
$ npm run all
```
